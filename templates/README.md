## 如何使用

### HTML模板 → Word/PDF
1. 用浏览器打开 .html 文件
2. 找到 "编辑以下内容" 注释处，修改姓名/经历/技能
3. 打印 → 另存为PDF（排版不变形）
4. 或全选复制 → 粘贴到Word/WPS编辑

### 面试宝典
- 面试前浏览对应分类，准备2-3个STAR故事
- 面试后回顾表现，记录面试官反馈
