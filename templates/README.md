## 如何使用

### 方法1：浏览器编辑（推荐）
1. 用浏览器打开对应的 HTML 文件
2. 右键 → "检查" 或按 F12 打开开发者工具
3. 在 Elements 面板中找到 `<!-- ====== 编辑以下内容 ====== -->` 和 `<!-- ====== 编辑结束 ====== -->` 之间的内容
4. 双击文字直接修改
5. 修改完成后 Ctrl+P → 另存为PDF

### 方法2：文本编辑器
1. 用 VS Code / 记事本 / Sublime 打开 HTML 文件
2. 搜索你要修改的内容（如"张三"）
3. 替换成你的信息
4. 保存后用浏览器打开 → 打印为PDF

### 方法3：导入Word/WPS
1. 浏览器打开HTML文件
2. 全选(Ctrl+A) → 复制
3. 粘贴到Word/WPS中
4. 调整格式后导出PDF

## ATS友好建议

- 使用标准字体（微软雅黑、宋体、Arial、Helvetica）
- 避免使用表格嵌套（本模板未使用）
- 关键词完整拼写，不要缩写
- 用PDF格式投递（排版不会乱）

## 常见修改项

1. 姓名、联系方式
2. 工作经历（公司名、职位、时间、描述）
3. 教育背景（学校、专业、时间）
4. 技能标签
5. 项目经验
